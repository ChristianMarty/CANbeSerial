# CANbeSerial

A protocol to transmit CAN and CAN-FD frames via a serial link (e.g. RS-232, TCP, UDP).

## Documentation
Documentation of the protocol can be found in the "documentation" folder of this project or [https://www.christian-marty.ch/CANbeSerial/documentation/index.html](https://www.christian-marty.ch/CANbeSerial/documentation/index.html)


## Implementation

An alternative firmware for the CANable 2.0 device can be found in the repository: [https://github.com/ChristianMarty/canable2-CANbeSerial-Firmware](https://github.com/ChristianMarty/canable2-CANbeSerial-Firmware)
