# CANbeSerial

A protocol to transmit CAN and CAN-FD frames via a serial link (e.g. RS-232, TCP, UDP).

## Documentation
Documentation of the protocol can be found as raw HTML in the "documentation" documentation folder of this project or is or [https://www.christian-marty.ch/# CANbeSerial
